# Geniatech DTV Bridge

Local Home Assistant app/add-on wrapper for `runawaysheep/Geniatech-DTV-Bridge`.

## Default options
- ip: `192.168.0.252`
- mode: `server`
- port: `30012`
- profile: `fhd`
- forceDeviceChange: `true`

After install, check logs first. If it starts correctly, follow the upstream project's channel scan / request flow.
